# Fancy Login Demo

A simple, modern single-page web application built using **HTML**, **CSS**, and **JavaScript**.

## Features

* 🎨 Animated gradient background
* ✨ Glassmorphism login card
* 👤 Simple login screen
* 🏠 Landing page displaying a personalized welcome message
* 🚀 "Get Started" button
* 📱 Responsive design
* ⚡ No frameworks or dependencies

## Getting Started

### Prerequisites

* Any modern web browser (Chrome, Edge, Firefox, Safari)

### Run the Application

1. Clone or download this repository.

2. Navigate to the project folder.

3. Open `index.html` in your browser.

That's it! No installation or web server is required.

---

## Usage

1. Enter any username.
2. Enter any password (validation is not implemented).
3. Click **Login**.
4. You will be redirected to the landing page displaying:

```
Welcome <Your Username>
```

5. Click **Get Started** to display a welcome message.
